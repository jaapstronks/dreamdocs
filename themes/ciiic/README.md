# CIIIC Theme

Professional document theme for CIIIC organization.

## Fonts

This theme uses **GT America** (proprietary) and **JetBrains Mono** (open source).

- GT America fonts are licensed and should not be redistributed
- When publishing this as open source, exclude this theme or replace fonts

## Colors

| Purpose | Color | Hex |
|---------|-------|-----|
| Accent | Lime Yellow | `#dbff00` |
| Primary | Teal | `#38766d` |
| Text | Near Black | `#141414` |
| Callout BG | Light Teal | `#e0e5e2` |

## Features

- Extended font family for headings (GT America Extended)
- Lime yellow accent borders on headings and callouts
- Teal table headers
- Professional typography with good line spacing

## Separation Note

This theme is marked as `"private": true` in theme.json to indicate it should be excluded when publishing the app as open source. To create a separable theme system:

1. Move theme to a separate repo/package
2. Allow theme installation via config
3. Document theme API for third-party themes
